====================================

GameComponent
  │
  ├─→ GameService ──→ Backend API (REST)
  │
  ├─→ WebSocketService ──→ Backend WebSocket
  │
  ├─→ AuthService ──→ Current User State
  │
  ├─→ UserService ──→ User Profile API
  │
  ├─→ RankingService ──→ Ranking API
  │
  └─→ BackgroundMusicService / GameResultSoundService ──→ Audio

GameUserProfileComponent
  │
  ├─→ UserService ──→ Update Avatar API
  │
  └─→ AuthService ──→ Current User State
