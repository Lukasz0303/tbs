# Verona Theme Enforcement

- Globalny styl aplikacji musi opierać się na motywie PrimeNG Verona z `https://verona.primeng.org/`.
- Nie dodawaj ani nie utrzymuj alternatywnych motywów PrimeNG, Tailwind UI czy innych bibliotek motywujących poza Tailwindem wspierającym układ.
- Wszystkie nowe komponenty, widoki i poprawki UI muszą być projektowane i testowane względem motywu Verona (zarówno w trybie jasnym, jak i ciemnym, jeśli dotyczy).
- W razie potrzeby aktualizacji stylu, aktualizuj centralny import w `frontend/src/styles.css`; nie nadpisuj motywu lokalnymi, doraźnymi stylami.
- Dokumentuj każdą zmianę styli w dokumentacji projektowej, aby zachować spójność motywu w całym produkcie.
